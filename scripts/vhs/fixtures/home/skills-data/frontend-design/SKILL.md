# Frontend Design

Create distinctive, production-ready user interfaces with modern CSS, responsive layouts, and thoughtful design systems.

## Features

- Component-based design architecture
- Responsive and accessible layouts
- Design system generation
- Tailwind CSS integration
- Dark/light mode support

## Usage

Describe the interface you want to build, and this skill generates production-ready HTML, CSS, and JavaScript components.
