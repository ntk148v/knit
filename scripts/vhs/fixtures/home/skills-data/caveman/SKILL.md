# Caveman Mode

A communication skill that strips filler, articles, and pleasantries while keeping full technical accuracy.

## Features

- Drops articles (a, an, the), linking verbs, and polite filler
- Retains all technical accuracy and specificity
- Cuts token usage by approximately 75%
- Activates on `caveman mode`, `talk like caveman`, or `/caveman`

## Usage

Invoke by saying "caveman mode" or "talk like caveman". The agent will respond in compressed form.
